SEO Attributes in HTML

What is SEO?

SEO (Search Engine Optimization) is the practice of improving a website to increase its visibility on search engines like Google.

SEO Attributes Used in This Page

  • <title>: Sets page title in browser and search results.
  • Meta Description: Short summary shown in search results.
  • Meta Keywords: List of keywords (less used now).
  • Meta Author: Name of the content creator.
  • Meta Robots: Tells search engines to index and follow.
  • Canonical Link: Prevents duplicate content issues.
  • Open Graph Tags: Controls how the page appears on social media.
  • Twitter Tags: Improves tweet previews.
  • JSON-LD: Adds structured data for rich results.
  • Alt Attribute: Describes images for SEO and accessibility.
